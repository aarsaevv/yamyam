# yamyam
Asian food delivery. For now this is frontend-only version. Works on Bootstrap 5 and vue.js 3. Layout was made on Figma.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```

```
